Git常用命令

发表于 2018-06-08 |

安装Git

在mac上使用homebrew安装
brew install git

##配置用户信息

$ git config --global user.name "username"
$ git config --global user.email example@example.com

查看配置信息

$ git config --list

初始化新仓库

$ git init

跟踪管理项目

$ git add file
$ git commit -m 'initial project version'

从现有仓库克隆

$ git clone git://github.com/schacon/grit.git

检查当前文件状态

$ git status

查看已暂存和未暂存的更新

$ git diff

移除文件

$ git rm file
$ git rm --cached readme.txt

移动文件（重命名）

$ git mv file_from file_to

查看提交历史

$ git log

修改最后一次提交

$ git commit --amend

版本回退

$ git reset --hard head^
$ git reset --hard id

撤销文件在工作区的修改

git checkout -- file
